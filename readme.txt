Using Gravity Forms - Categories WordPress Plugin the admin can categorize Gravity Forms.

== Description ==
1. Using Gravity Forms - Categories WordPress Plugin the admin can categorize Gravity Forms.
2. Admin can create new categories for gravity forms.
3. From the form settings, Admin can assign multiple categories to that form.
4. Filter forms based on the categories assigned. 

== Installation ==

1. Copy the entire /gravityforms-categories/ directory into your /wp-content/plugins/ directory.
2. Activate the plugin.
3. New Tab "Categories" under the Forms will generate.
4. Perform Action.
